# Robust Secure Data Transmission Using AI in MANET

This project applies Artificial Intelligence techniques to ensure secure and reliable data transmission within Mobile Ad-Hoc Networks (MANETs).

# Project Files
* `RobustSecureData.py` / `ABC.py` - Core project scripts and AI algorithms.
* `test.py` - Script to run tests and evaluate performance.
* `testData.csv` / `AODV.csv` - Datasets used for testing network traffic.
* `run.bat` - Simple batch file to run the project on Windows.

# How to Run

# Windows 
Simply double-click the **`run.bat`** file to launch the project.

# Command Line
Open your terminal in this folder and run:
```bash
python test.py
